# DHalgorithm
Программа для демонстрации работы протокола Диффи-Хеллмана и Алгоритма Поллига-Хеллмана.

Для запуска и компиляции программы необходимо установить библиотеку GMP.

Программа запускается с определенными флагами. Флаг -Normal означает, что программа будет оперировать с длинными числами, которые не годятся для ручной проверки результатов работы программы, и служат лишь для демонстрации того, как протокол работает в реальности. Флаг -Demo, напротив, служит для того, чтобы программа оперировала с небольшими числами, что позволяет достаточно легко проверить результаты работы программы.
